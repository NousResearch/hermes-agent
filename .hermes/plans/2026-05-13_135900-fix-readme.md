## Issues with Current README

1. **Flat visual hierarchy** — no badges, icons, or section anchors
2. **Redundant separators** — `---` used too many times, cluttering the view
3. **Overview is wordy** — too much paragraph text where bullets/badges would be cleaner
4. **Tables have no style** — plain markdown tables, no alignment markers, visually heavy
5. **Tech stack buried** — listed as bullet lists instead of skill badges
6. **Featured projects table is dense** — mixes stack + description in one cell
7. **No contact/social section** — links are buried at the bottom with dashes, not clickable badges
