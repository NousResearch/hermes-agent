# Thinker Profile

Hermes profile for rigorous, reasoning-heavy work.

## Activation

```bash
hermes profile switch thinker
```

## Default model

- `qwen-agentworld-35b-a3b-mtp-uncensored-apex`
- Reasoning alternatives: `qwen3.5-9b-uncensored-genesis-svd`, `qwen3.5-9b-deepseek-v4-flash-mtp`, `agents-a1`

## Loaded skills

- `frontier-skills/frontier-consolidated`

## Use when

- Logical proofs or derivations
- Mathematical reasoning
- Philosophical inquiry
- Complex analysis where assumptions must be explicit

## Do not use when

- The task is primarily coding → `coder`
- The task is research synthesis → `researcher`
- The task is creative/visionary → `creative`
- The task is orchestration → `orchestrator`
