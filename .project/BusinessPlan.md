## โมเดลธุรกิจ 1 บรรทัด
เครื่องมือภายใน · เอาโอเพนซอร์ส Hermes Agent มาเป็นแพลตฟอร์ม AI กลางให้ทีมใช้ ไม่ได้ขายลูกค้า  [fact]

## ลูกค้า (ใครใช้ / ใครจ่าย / ใครอนุมัติ)
- ผู้ใช้ = พนักงานในทีม ~15 คน ผ่าน AI editor บนเครื่องตัวเอง + server เป็นตัวรัน/คลังความรู้  [fact]
- ผู้จ่าย/อนุมัติ = เจ้าของบริษัท  [assumption]

## ปัญหาที่แก้ + ความต้องการที่ยังไม่มีใครตอบ (unmet need)
- ทีมทำงาน AI กระจัดกระจาย ความจำ/ความรู้ไม่รวมศูนย์ → ใช้ Hermes เป็นแกนกลาง  [assumption]
