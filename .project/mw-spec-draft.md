## 2. ทางเข้า: โหมดหลัก + แทร็กเสริม (แก้ขัดแย้ง Codex ข้อ 1)

- โหมดหลัก: `MIGRATE` / `REMEDIATE` / `BUILD` · แทร็กเสริม: `DATA` / `FORM` / `MINISITE` / `BILINGUAL`
- **นิยามใหม่ (ไม่มีคำว่า "ข้ามขั้น"):** โหมด**กำหนดรูปของขั้น** ไม่ตัดขั้น — ทุกโหมดเดินครบ 13 สถานี:
  - M1: MIGRATE = capture เว็บเก่า · REMEDIATE = **M1R** audit เนื้อหาจริงเว็บปัจจุบัน (กัน soft-404 — ตรวจ "คำที่ควรมี" ไม่ใช่ HTTP 200) · BUILD = **M1-D** inventory+mapping ฐานข้อมูล
  - M0.5 Benchmark Gate: มี input = บังคับเต็ม · ไม่มี input = ออกผล `NO_INPUT` ลงบัตรเมนู (ไม่ใช่ "ข้าม" — มีร่องรอยเสมอ)
- profile.yaml ล็อกโดย owner · AI เปลี่ยนโหมด/เพิ่มแทร็กเอง**ไม่ได้** — เจองานนอกโหมด → เสนอเพิ่มแทร็ก รอ owner อนุมัติ
