# Commercial support

Hermes Agent is open source under the MIT license. Teams that want help deploying, extending, or operating Hermes can contact Nous Research for commercial support.

Common engagements include:

- self-hosted Hermes deployments for engineering, operations, or research teams
- Slack, Discord, Telegram, email, or API Gateway setup
- MCP server/client integration work
- custom tools, skills, plugins, and workflow automation
- security review, approval workflows, and production runbooks
- model/provider configuration and cost optimization

Start with the community links in the README, or contact Nous Research through the channels listed at [nousresearch.com](https://nousresearch.com).

Commercial support is optional. The open-source project remains fully usable without a support contract.
