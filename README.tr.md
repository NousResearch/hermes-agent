# Hermes Agent 🦅

**Hermes Agent**, [Nous Research](https://nousresearch.com/) tarafından geliştirilen, kendi kendini geliştirebilen (self-improving) bir yapay zeka ajanıdır. 

Oluşturduğu "öğrenme döngüsü" sayesinde deneyimlerinden yeni yetenekler yaratır, kullanım sırasında kendini iyileştirir ve oturumlar boyunca sizin kim olduğunuza dair derinleşen bir model oluşturur.

## Özellikler
* Bulut tabanlı çalışabilme (5 dolarlık bir VPS veya sunucusuz altyapı).
* Telegram üzerinden iletişim kurabilme yeteneği.
* Kendi hafızasını ve geçmiş konuşmalarını arayabilme.
