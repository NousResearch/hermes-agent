Hermes Agent v0.13.109

This Android release adds the compact Hermes chat UI pass and theme customization controls requested for mobile use.

- Added compact and expanded chat display modes, persisted in app settings.
- Added customizable theme colors plus rounded, soft, and square card shape controls.
- Improved chat readability with collapsed prompts, quieter timestamps and tool rows, focused image previews, and a secondary scroll-to-bottom affordance.
- Preserved attached image metadata in stored conversations and native chat history.
- Fixed sequential native tool-call finalization after Android browser handoffs.
- Validated the Android UI, native tool loop, Gemma LiteRT-LM, and Qwen GGUF emulator paths.
