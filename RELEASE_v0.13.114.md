# Hermes Agent Fork v0.13.114

This release publishes the Android fork under its new application identity:
`com.mobilefork.hermesagent`.

## Android

- Rebrands the installed app and F-Droid listing as `Hermes Agent Fork`.
- Keeps fork status visible in app metadata, settings, and top status badges.
- Publishes signed universal APK and AAB artifacts for F-Droid reproducible-build
  verification and future Play Store submission preparation.
- Keeps F-Droid update discovery future-facing through the tagged
  `fdroid/com.mobilefork.hermesagent.version` file.
