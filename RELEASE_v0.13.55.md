## Tasker Flash Toast Automations

This release adds a safe Tasker Flash parity slice for Hermes Android.

- Adds `create_toast_task` and `show_toast` automation actions.
- Adds `create_flash_task` / `flash` aliases so model-created actions can use Tasker terminology.
- Imports safe Tasker XML Flash actions into disabled-by-default Hermes toast records.
- Expands saved Hermes variables at run time and caps toast message text.
