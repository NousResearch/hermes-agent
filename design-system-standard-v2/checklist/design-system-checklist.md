## 🔴 กฎ SERIOUS · เหนือทุกชั้น (แหก = ปิดงานไม่ได้)

### S1 · ห้ามงานดูเป็น AI (Anti-AI-Slop) · Serious Case
งานที่ส่งต้อง **ดูเหมือนคนออกแบบ** ไม่ใช่ค่าตั้งต้นที่ AI พ่นออกมา · เป็นกฎบล็อก ไม่ใช่คะแนน · **รายละเอียดเต็ม `spec/07-anti-ai-slop.md`**

| สัญญาณกลิ่น AI (❌ ห้าม) | สิ่งที่คนออกแบบทำ (✅ ทำแทน) |
|---|---|
| ฟอนต์ Inter/Geist ทุกที่ (ค่าตั้งต้น AI) | คู่ฟอนต์ที่เลือกเองให้เข้าแบรนด์ + จังหวะขนาดเฉพาะ |
| ไล่สีม่วง-คราม (purple→indigo gradient) | สีแบรนด์จริงจากด่านสี (OKLCH/LCH derive เอง) |
| กล่องมน `rounded-lg` + เงานุ่มเท่ากันทุกใบ | radius/เงา ที่ตั้งใจให้ต่างตามลำดับความสำคัญ |
| การ์ดเท่ากันเป๊ะเต็มจอ สมมาตรตายตัว | จังหวะ/ลำดับสายตาตั้งใจ (มี asymmetry มีพระเอก) |
| ภาพ stock/ภาพ AI ลอย ๆ ไม่มีบริบท | ภาพจริงของงาน/ไอคอน/กราฟิกเฉพาะโครงการ |
| หัวข้อแปะ emoji + คำโปรย seamless/elevate/robust | ถ้อยคำเสียงแบรนด์จริง (ดู C12) |
| ผิวเรียบเกลี้ยงไม่มีพื้นผิว | เพิ่ม grain/noise/texture บาง ๆ ให้รู้สึกมีมือคน |
| ไม่มี interaction เฉพาะตัว | micro-interaction ที่มี concept (ดู U1/A19) |

**เกณฑ์ผ่าน S1:** งานต้องไม่ติดสัญญาณ ❌ เกิน 1 จุด + รัน `Use Impeccable` แล้วผ่าน → ถึงปิดงานได้
**บังคับที่ 3 จุด:** (1) กฎ Serious ข้อนี้ (2) ด่าน D12 ตอนปิดงาน (3) กำหนด "ลายเซ็นความเป็นคน" ในชั้น F/H ก่อนออกแบบ
