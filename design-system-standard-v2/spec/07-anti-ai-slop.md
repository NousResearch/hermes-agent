# สเปค S1 · Anti-AI-Slop — ห้ามงานดูเป็น AI (SERIOUS · v3 · 2026-07-12)

> **กฎ SERIOUS · เหนือทุกชั้น · แหก = ปิดงานไม่ได้** (owner สั่ง 2026-07-12)
> งานที่ส่งต้อง "ดูเหมือนคนออกแบบ" ไม่ใช่ค่าตั้งต้นที่ AI พ่นออกมา
> ตัวรันตรวจ = `Use Impeccable` · คู่กับเช็กลิสต์ D12

## ทำไมเป็น Serious Case
กรรมการ/ลูกค้าเปิดงานมาแล้วรู้สึก "นี่มัน AI ทำ" = เสียความน่าเชื่อถือทันที · งานราคาหลายล้านต้องดูมีฝีมือคน · เดิมเรื่องนี้เป็นแค่คะแนน 5% → ยกเป็นกฎบล็อก

## รากของปัญหา (distributional convergence)
AI ทุกตัวถูกเทรนจากข้อมูลคล้ายกัน → พ่นหน้าตาซ้ำ ๆ ชุดเดียว: ฟอนต์ Inter + ไล่สีม่วง-คราม + กล่องมน rounded-lg + เงานุ่ม + การ์ดสมมาตร · พอเห็นชุดนี้ = คนจับได้ว่า AI ทำ

## ตารางบังคับ · สัญญาณกลิ่น AI (❌) vs สิ่งที่คนทำ (✅)

| สัญญาณกลิ่น AI (❌ ห้าม) | สิ่งที่คนออกแบบทำ (✅ ทำแทน) | เครื่องมือ/วิธี |
|---|---|---|
| ฟอนต์ Inter/Geist ทุกที่ | คู่ฟอนต์เลือกเองให้เข้าแบรนด์ + จังหวะขนาดเฉพาะ | Utopia fluid type · type pairing |
| ไล่สีม่วง-คราม (purple→indigo) | สีแบรนด์จริงจากด่านสี derive เอง | OKLCH/LCH · Huetone |
| rounded-lg + เงานุ่มเท่ากันทุกใบ | radius/เงาตั้งใจต่างตามลำดับความสำคัญ | token elevation ที่ออกแบบ |
| การ์ดเท่ากันเป๊ะ สมมาตรตายตัว | จังหวะ/ลำดับสายตาตั้งใจ (asymmetry · มีพระเอก) | ดู U3/U4 |
| ภาพ stock/ภาพ AI ลอย ๆ | ภาพจริงของงาน/ไอคอน/กราฟิกเฉพาะ | asset production จริง |
| หัวข้อแปะ emoji + seamless/elevate/robust | ถ้อยคำเสียงแบรนด์จริง | ดู C12 คลังคำ |
| ผิวเรียบเกลี้ยงไม่มีพื้นผิว | grain/noise/texture บาง ๆ ให้รู้สึกมีมือคน | noise overlay |
| ไม่มี interaction เฉพาะตัว | micro-interaction ที่มี concept | GSAP · Rive · ดู A19 |

## บังคับ 3 จุด (ป้องกันเป็นชั้น)
1. **กฎ Serious บนสุด** — ประกาศเจตนา · ทุกงานยึด
2. **ด่าน D12 ตอนปิดงาน** — รัน `Use Impeccable` + ตรวจตารางข้างบน
3. **ลายเซ็นความเป็นคน (ชั้น F/H)** — กำหนด "จุดที่ทำให้งานนี้ดูมีมือคน" ตั้งแต่ต้นทาง (บันทึกใน H7) ไม่ใช่มาไล่ลบทีหลัง

## เกณฑ์ผ่าน (หลักฐาน)
- รัน `Use Impeccable` แล้วผ่าน (แนบผล)
- ตรวจตารางสัญญาณ → **ติด ❌ ได้ไม่เกิน 1 จุด**
- ระบุ "ลายเซ็นความเป็นคน" ของงานนี้ ≥ 2 อย่าง (เช่น คู่ฟอนต์เฉพาะ + noise texture + micro-interaction ที่มี concept)
- ไม่ผ่านข้อใดข้อหนึ่ง = **ปิดงานไม่ได้** (Serious)

## หมายเหตุ
ไม่ได้ห้ามใช้ AI ช่วยทำ — ห้าม "ผลลัพธ์ที่ดูเป็นค่าตั้งต้น AI" · ใช้ AI ทำได้ แต่ต้องดันให้พ้นชุดหน้าตาซ้ำ ๆ จนดูมีฝีมือคน
