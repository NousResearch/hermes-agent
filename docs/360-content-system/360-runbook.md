# คู่มือเปิดใช้ระบบ 360 — Hi Logic Labs

> ระบบสร้างเสร็จและตรวจแล้วทุกส่วนที่ทำได้โดยไม่ต้องกุญแจ
> ไฟล์นี้บอกวิธีกดใช้จริงทีละขั้น และระบุงานที่พี่นัทต้องทำเอง

## สายงานทั้งหมด (ทบทวนสั้น)

แชท AI → การ์ดความรู้ → บล็อก → LinkedIn/Facebook/สไลด์/พอดแคสต์/วิดีโอ → เผยแพร่
ทุกชิ้นผ่านด่านข้อมูลลับก่อนออก

## ขั้นกดใช้จริง (เรียงตามลำดับ)

| ขั้น | คำสั่ง | อยู่ที่ | สถานะ |
|---|---|---|---|
| 1. เก็บเกี่ยวแชท | `python scripts/harvest_chat_knowledge.py --source claude-code` | Hermes | ใช้ได้เลย |
| 2. ตรวจ/อนุมัติการ์ด | เปิดดู `docs/360-content-system/knowledge-inbox/_new/` แล้วย้ายใบที่ผ่านไป `_approved/` | Hermes | งานคน (อนุมัติ) |
| 3. ผลิตเนื้อหา | `node scripts/produce-content.mjs <blog.md>` | Content Factory | รอกุญแจ AI |
| 4. ตรวจข้อมูลลับ | `node scripts/privacy-gate.mjs <ไฟล์>` | Content Factory | ใช้ได้เลย |
| 5. กระจายแพลตฟอร์ม | ตัวเชื่อมใน `src/services/social/` (linkedin, facebook, medium, devto) | Content Factory | รอ token + 2FA |
| 6. ตรวจโปรเจกต์อัปเดต | `python scripts/project_sync_watcher.py` | Hermes | ใช้ได้เลย |

## งานที่พี่นัทต้องทำเอง (คน · ทำครั้งเดียว)

1. **จดโดเมน** `hilogiclabs.com` ที่ผู้รับจดโดเมน
2. **ใส่กุญแจ AI** ในไฟล์ `.env` ของ Content Factory:
   - `OPENROUTER_API_KEY` — เพื่อให้ AI เขียนเนื้อหา (ขั้น 3)
   - `MINIMAX_API_KEY` — เพื่อทำเสียงพูด + วิดีโอ (เฟส 4.5)
3. **ขอ token แพลตฟอร์ม** (LinkedIn / Facebook / Medium) + ผ่าน 2FA เพื่อโพสต์อัตโนมัติ (ขั้น 5)
