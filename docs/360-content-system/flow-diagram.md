## 3 ข้อที่คนมักสงสัย

1. **พี่ลงแรงแค่ 3 จุด** (`[●]`): สั่งเก็บเกี่ยว · อนุมัติการ์ด · อนุมัติชิ้นสุดท้าย — ที่เหลือระบบทำเอง
2. **ด่านข้อมูลลับมี 2 ชั้น** (ขาเข้า + ก่อนปล่อย) กันชื่อบริษัท/ลูกค้าหลุดสองรอบ
3. **ทุกจุด `[$]`** คือที่ใช้กุญแจ — ใส่กุญแจเมื่อไรขั้นนั้นผลิตจริงทันที ยังไม่ใส่ก็เดิน dry-run ครบสาย
