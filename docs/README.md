## Federated Apps Overview

The Federated Apps system (Slice 06: Normalized State Artifact) enables distributed applications to:

- Register themselves with the federation
- Expose operators (capabilities/providers)
- Establish wiring connections between operators across app boundaries
- Track connection states and health status

See [federated-apps-api-data-contract.md](./federated-apps-api-data-contract.md) for complete API documentation.
