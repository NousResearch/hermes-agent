Brand Snapshot

The System is the restrained premium default brand used when a coach-specific identity has not yet been approved. It should feel structured, intelligent, useful, and high-trust without overpowering the coach.

Role: default house-brand wrapper for early coach-agent outputs.
Posture: premium editorial, calm, precise, operationally sharp.
Boundary: fallback system, not a replacement for coach identity.
Quick tokens

Identity: stacked wordmark, TS monogram, combined lockup.

Color: deep green, warm cream, restrained gold.

Type: editorial serif display + clean sans body + mono metadata.

Voice: useful, calm, premium, specific.

Source status

Logo assets and default brand role are source-backed from operator-approved materials.

Downstream use

Premium lesson recaps, slide templates, onboarding artifacts, default documents.

Design rule

Use the brand as quiet structure: chrome, hierarchy, trust — never decoration for its own sake.

Typography

Typography should read like a premium private-client brief: editorial serif for hierarchy, clean sans for clarity, monospaced labels for operational metadata.

Heading token
Structure creates trust.

Family: Playfair Display / Georgia fallback · Weight: 600–700 · Cover: 88–98px / .95 · Section: 54–60px / 1.0 · Use: cover titles, section titles, premium dividers.

Body token
A calm, direct, useful explanation style for coaches and students.

Visual Style

The System visual style is a premium operating layer: structured, quiet, useful, and refined. It should create confidence through clarity, not impress through noise.

Page rhythm: large margins, clear cards, visible hierarchy.
Chrome: thin gold rules, green mastheads, cream panels.
Texture: subtle editorial warmth; no generic AI stock gloss.
Density: enough substance to be useful, never cluttered.
Do / don't

Do: make artifacts feel like a serious private-client brief.

Do: leave room for coach-specific brand replacement.

Don’t: use gradients as decoration, cartoon sports marks, or over-designed luxury cues.

Don’t: let house branding overpower the coach’s expertise.

“The brand should make the work feel structured, premium, and supported — not branded for branding’s sake.”
