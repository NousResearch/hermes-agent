## Ethical Governance Framework for Contributions

All contributions to the Burgess Principle integration within Hermes Agent are governed by the Burgess Principle itself.

### The Standard

The Burgess Principle establishes a binary test for ethical accountability:

> **"Was the individual considered as an individual human being, or were they processed as a unit within a system?"**

This test applies not only to the software's behaviour but also to the process of building it.

### How This Applies to Contributors

When contributing code, documentation, skills, or tools that relate to the Burgess Principle:

1. **Features that affect individuals must implement the binary test.** Any new feature, skill, or tool that acts on or about an individual must include the Burgess Principle check as a pre-action step. See [INTEGRATION.md](INTEGRATION.md) for the integration pattern.

2. **Do not weaken the binary nature of the test.** The test produces PASS or FAIL. Do not introduce graduated scoring, partial compliance, or "soft pass" concepts. The binary nature is a deliberate design choice that prevents organisations from claiming partial compliance.

3. **Maintain auditability.** Contributions must preserve the ability to produce a clear audit trail when the binary test is applied. Every test invocation should record the decision description, the result, and the reasoning.

4. **Respect the licensing structure.** The four-tier licensing model (see [LICENSING.md](LICENSING.md)) is integral to the framework. Do not modify licensing terms or introduce alternative licensing without authorisation from the framework author.

5. **Attribution is required.** All materials referencing the Burgess Principle must attribute Lewis James Burgess as the author and link to the canonical source at [github.com/ljbudgie/burgess-principle](https://github.com/ljbudgie/burgess-principle).

### Contribution Process

- Follow the standard Hermes Agent contribution process as documented in the repository root `CONTRIBUTING.md`.
- For changes specific to the Burgess Principle integration, ensure they align with the framework's principles as described in [OVERVIEW.md](OVERVIEW.md).
- When in doubt about whether a contribution aligns with the Burgess Principle, apply the binary test to your own contribution: does it treat individuals as individuals, or does it enable processing them as units?
