## Safe rules

- One family, one profile.
- No shared cookies.
- No passwords, backup codes, or live credentials in repo docs.
- No extra business accounts without a real purpose, owner, recovery path, profile, and trust level.

## If you are unsure

Start with the segmentation map, then the profile map, then the registry draft. If the account has no real job, leave it out of business.
