# Adaptation notes — ep001

## Compression rules
- 내면 독백은 캡션으로 축약
- 혐관 텐션은 얼굴 거리와 손/시선으로 강조
- 입시/장학/학생증/성적표 같은 소품으로 계급감을 시각화

## Scroll rhythm
- p01: opening / 압박 / 도서관 자리 위 약대 입결표와 반수 루틴 파일이 열린 노트북 클로즈업
- p02: character_intro / 경계 / 스터디실에 처음 들어온 윤서하와 노트북을 급히 돌리는 강시윤
- p03: status_hit / 수치 / 학교를 말한 직후 서하의 짧은 “아.”와 굳은 시윤 표정
- p04: conflict / 도발 / 서하가 반수와 연애를 연결해 말하는 장면
- p05: private_aftershock / 잔상 / 엘리베이터 거울에 비친 두 사람의 어색한 거리
- p06: message_reveal / 들킴 / 원룸 어둠 속 휴대폰 화면과 메모 캡처
- p07: family_pressure / 압박 / 엄마 메시지와 캡처 화면이 동시에 뜬 휴대폰
- p08: cliff / 예고 / 침대 끝에 앉아 화면을 보는 시윤의 실루엣
