# Adaptation notes — ep002

## Compression rules
- 내면 독백은 캡션으로 축약
- 혐관 텐션은 얼굴 거리와 손/시선으로 강조
- 입시/장학/학생증/성적표 같은 소품으로 계급감을 시각화

## Scroll rhythm
- p01: opening / 초조 / 과외 중 멍하니 있는 시윤과 휴대폰 답장 화면
- p02: duel / 탐색 / 비 오는 밤 스터디실에서 마주 앉은 두 사람
- p03: ego_hit / 도발 / 서하의 정면 클로즈업
- p04: counter / 분노 / 서하에게 쏘아붙이는 시윤
- p05: third_party / 긴장 / 조민우가 끼어들며 반수 느낌 나는 애라고 비웃는 장면
- p06: umbrella / 정전 / 빗속 우산 아래 가까워진 거리
- p07: touch / 흔들림 / 서하가 시윤 손목을 잡는 장면
- p08: cliff / 감시 / 건너편에서 휴대폰 카메라가 둘을 잡는 실루엣
