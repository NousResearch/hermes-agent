# Adaptation notes — ep003

## Compression rules
- 내면 독백은 캡션으로 축약
- 혐관 텐션은 얼굴 거리와 손/시선으로 강조
- 입시/장학/학생증/성적표 같은 소품으로 계급감을 시각화

## Scroll rhythm
- p01: opening / 확산 / 단톡방으로 퍼진 우산 사진 캡처
- p02: proposal / 거래 / 카페 구석에서 계약하듯 마주 앉은 두 사람
- p03: terms / 계산 / 서로 조건을 교환하는 손과 노트북 메모
- p04: routine / 중독 / 도서관과 카페를 오가며 자연스러워지는 페이크 연애 몽타주
- p05: public_scene / 과시 / 사람들 앞에서 팔을 잡고 데리러 와달라 말하는 서하
- p06: witness / 균열 / 복도 끝에서 장면을 목격한 소라
- p07: message / 잔향 / 밤에 오가는 메시지와 허리 감각 회상
- p08: cliff / 삼각 / 소라의 음성메시지 파형
