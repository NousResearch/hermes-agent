# Adaptation notes — ep005

## Compression rules
- 내면 독백은 캡션으로 축약
- 혐관 텐션은 얼굴 거리와 손/시선으로 강조
- 입시/장학/학생증/성적표 같은 소품으로 계급감을 시각화
