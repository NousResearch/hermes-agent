# 웹툰 어댑테이션 노트

## 어댑테이션 원칙
- 페이지 만화가 아니라 세로 스크롤 리듬으로 설계했습니다.
- 1화의 기능은 세계관 설명이 아니라 **비밀 들킴**입니다.
- 오르비 감정시장에 맞게 성취 서사보다 위치 추락 공포를 전면에 뒀습니다.

## 이 에피소드가 먹히는 이유
1. 입결표/계약학과/반수라는 현재 학생 언어가 그대로 들어갑니다.
2. 주인공이 선/악이 아니라 `상방 욕망 + 하방 계산`의 혼합체라 계속 굴릴 수 있습니다.
3. 2화 이후 확장 포인트가 분명합니다.
   - 담임 상담
   - 스터디방 유출
   - 아버지 개입
   - 약대로 목표 하향 유혹
   - 6모 직전 과탐 붕괴

## 실제 렌더 주의
- 방/노트북/책상 앵커를 먼저 잡고 edit chain으로 이어야 컷 연속성이 살아납니다.
- 한글 텍스트는 이미지 생성에 맡기지 말고 식자로 후처리해야 합니다.
- 사용자 선호에 맞춰 최종 산출은 `1 episode = 1 PNG long scroll`로 합치면 됩니다.
