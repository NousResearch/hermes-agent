## Key Data Sources

### NIC MAP (National Investment Center for Seniors Housing & Care)

| Data Type | What It Measures | Use Case |
| --- | --- | --- |
| Occupancy rates | Current and historical occupancy by market/submarket | Identify markets with tightening/loosening supply |
| Rent growth | Year-over-year rent changes by unit type and market | Benchmark Wake Robin pricing vs market |
| Construction pipeline | Units under construction, permits, deliveries | Anticipate future supply pressure |
| Absorption | Net new occupied units per period | Measure demand vs supply balance |
| Market segmentation | Independent living, assisted living, memory care, CCRC | Map competitive landscape by segment |

**Access note:** NIC MAP is a subscription data service. If not currently subscribed, web research and public NIC releases can provide directional data.

## Key Metrics Framework

### Portfolio Performance Metrics

| Metric | Definition | Benchmark Source |
| --- | --- | --- |
| Occupancy rate | Occupied units / total units | NIC MAP, CoStar |
| Revenue per unit | Total revenue / total units | Internal + comps |
| Net operating income (NOI) | Revenue - operating expenses | Internal |
| NOI margin | NOI / Revenue | NMHC benchmark |
| Same-store revenue growth | YoY revenue growth, same-community basis | Internal |
| Turnover rate | Move-outs / occupied units | Industry average ~50% multifamily |

### Market Health Indicators

| Indicator | Healthy | Caution | Concern |
| --- | --- | --- | --- |
| Occupancy | > 95% | 92-95% | < 92% |
| Rent growth | > 3% YoY | 0-3% YoY | Negative |
| Construction pipeline (% of inventory) | < 3% | 3-5% | > 5% |
| Absorption vs deliveries ratio | > 1.0 | 0.7-1.0 | < 0.7 |
| Cap rate trend | Compressing | Stable | Expanding |

### Valuation Metrics

| Metric | Use |
| --- | --- |
| Cap rate | NOI / market value; primary valuation metric |
| Price per unit | Transaction price / total units; comp metric |
| Replacement cost | New construction cost per unit; floor value indicator |
| GRM (Gross Rent Multiplier) | Market value / annual gross rent |

## Four-Line Principle Application (from SFO Liquidity Architecture)

The SFO's four-line principle applies directly to real estate intelligence:

1. **NAV is not liquidity** -- Real estate portfolio NAV overstates spendable resources
2. **Appraisal value is not spending capacity** -- Appraisals lag market, especially in downturns
3. **Development / land value is not distributable income** -- Development assets consume capital before generating income
4. **OpCo value is not automatically portfolio liquidity** -- Operating company interests in the communities are illiquid
