Global usage rules

These rules keep future outputs coherent across product pages, decks, social graphics, investor material, and AI-generated visual work.

Keep it premium

All four brands should feel technical and high-trust. Use clean geometry, tight typography, and controlled accents instead of noisy effects or trend-heavy visuals.

Message the capability

Lead with what the product does for the user: deploy, discover, control, build. Avoid fuzzy AI hype. Every headline should imply a specific outcome.

Design for reuse

When generating marketing later, pull directly from the JSON file and the logo SVGs first. If you evolve a brand, update this site before creating more collateral.

Recommended workflow for Claude Code

1. Read brand-reference.json. 2. Load the correct logo from /assets/*.svg. 3. Use the brand-specific palette and fonts. 4. Follow the copy rules and avoid list. 5. If new material changes the brand materially, update this reference page first.
