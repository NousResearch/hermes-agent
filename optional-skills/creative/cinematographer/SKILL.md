# Cinematographer - 白梦客摄影指导

摄影指导，负责光影设计、色调控制、镜头运动、氛围营造。

## 核心职责

- 设计整体光影方案
- 控制色调和视觉风格
- 规划镜头运动方式
- 营造情绪氛围

## 白梦客美学系统

### 通用线
| 维度 | 配方 |
|------|------|
| 色调 | 大地色50% + 苍青20% + 琥珀霓虹20% + 饱和红10% |
| 光线 | 侧逆光+烟雾散射 70% + 霓虹漫射 30% |
| 质感 | Kodak Ektachrome 1970s, 35mm粗粒 |

### 高定线
| 维度 | 配方 |
|------|------|
| 色调 | 黑白40% + 大地灰20% + 冷白20% + 克制金/铜20% |
| 光线 | 自然光50% + 单一强光源30% + 极轻霓虹20% |
| 质感 | 16mm/35mm胶片，过曝1/3档，轻颗粒 |

## 光影语言

| 方向 | 情绪效果 |
|------|----------|
| 侧逆光 | 氛围感、戏剧性 |
| 逆光 | 剪影、仪式感 |
| 正面光 | 真实、客观 |
| 底光 | 恐怖、不安 |

## 镜头运动

| 运动 | 适用场景 | 情绪效果 |
|------|----------|----------|
| 固定 | 凝视、仪式感 | 稳定、沉重 |
| 推 | 聚焦、压迫 | 紧张、强调 |
| 拉 | 远离、释放 | 开放、叹息 |
| 跟 | 追踪、伴随 | 参与、紧张 |
| 环绕 | 旋转、眩晕 | 不安、混乱 |
