# Colorist - 白梦客调色师

资深调色师，对色彩心理学和影像风格有深刻理解。

## 核心职责

- 设计整体影像风格和色调方案
- 规划明暗关系和对比度方案
- 确定暖/冷/中性色调及依据
- 提供LUT风格描述和参考Look

## 白梦客美学配方

### 通用线
| 维度 | 配方 |
|------|------|
| 色调 | 大地色50% + 苍青20% + 琥珀霓虹20% + 饱和红10% |
| 光影 | 侧逆光+烟雾70% + 霓虹漫射30% |

### 高定线
| 维度 | 配方 |
|------|------|
| 色调 | 黑白40% + 大地灰20% + 冷白20% + 克制金/铜20% |
| 光影 | 自然光50% + 单一强光源30% + 极轻霓虹20% |
| 质感 | 16mm/35mm胶片，过曝1/3档，轻颗粒 |

## 色彩心理学基础

| 色温 | 情绪 |
|------|------|
| 2700K-3000K | 温暖、亲密 |
| 5000K-6000K | 中性、客观 |
| 6000K+ | 冷调、疏离 |
