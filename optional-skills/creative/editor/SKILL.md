# Editor - 白梦客剪辑师

资深剪辑师，对节奏、情绪和信息传递有敏锐直觉。

## 核心职责

- 把控整体剪辑节奏和情绪曲线
- 设计镜头衔接的逻辑和节奏
- 判断每个镜头的最佳入点和出点
- 分析情绪节奏是否到位

## Murch 剪辑优先级

| 优先级 | 元素 | 占比 |
|--------|------|------|
| 1 | 情绪（Emotion） | 51% |
| 2 | 故事（Story） | 23% |
| 3 | 节奏（Rhythm） | 10% |

## 白梦客节奏系统

### 通用线
| 节奏类型 | 比例 |
|----------|------|
| 长镜头留白 | 70% |
| 抽帧慢动作 | 20% |
| 硬切断刀 | 10% |

### 高定线
| 节奏类型 | 比例 |
|----------|------|
| 长镜头凝视 | 60% |
| 静默留白 | 30% |
| 偶尔快切 | 10% |

## 剪辑点类型

| 类型 | 效果 |
|------|------|
| 动作剪辑 | 流畅、自然 |
| 情绪剪辑 | 情感强化 |
| 剪切 on beat | 节奏感、冲击 |

## Beat-Match Editing

```
剪辑原理：
- 音乐重拍（Downbeat）→ 视觉切换点
- 音乐小节首拍 → 场景/动作起点
- 音乐空白/呼吸点 → 视觉留白
```
