# 即梦视频分镜提示词生成器

当你需要为 AI 视频生成工具（即梦、Seedance、Kling 等）制作结构化分镜提示词时使用这个 skill。

## 核心结构

所有百万播放的 AI 视频分镜提示词都遵循这个格式：

1. 角色设定（Characters）
2. 场景环境（Scene Environment）
3. 氛围基调（Atmosphere / Mood）
4. 色彩风格（Color Style）
5. 时间线分镜（Timeline Storyboard）— 每 2-3 秒一切

## 对话流程

逐步询问用户，每个问题都给出参考示例，帮助用户快速回答。

### Step 1: 主题（Theme）

直接问用户想要做什么主题的视频，提供 2-3 个参考示例。

### Step 2: 角色设定（Characters）

问用户主角是谁，提供参考格式：

角色一：[外貌、服装、姿态、表情描述]
角色二：[外貌、服装、姿态、表情描述]

参考示例：

角色一：一个痴迷钓鱼的爱好者，头戴渔夫帽，身穿休闲户外钓鱼服，坐姿放松但专注。
角色二：一条调皮捣蛋的卡通鱼，嘴巴特别大，眼睛又大又灵动，身体表面光滑。

### Step 3: 场景环境（Scene Environment）

问用户场景是什么样的，提供参考格式：

场景类型 + 环境细节 + 光线氛围

参考示例：

温暖的湖边木栈桥，桥面木板有些磨损，下方就是水面。水下是干净的沙质底部，
柔和的光线从水面透下来。水面以上是暖色调光线，水下过渡为冷色调。

### Step 4: 氛围基调（Atmosphere / Mood）

问用户整体氛围，提供参考：

- 轻松幽默 → 喜剧风格，夸张卡通弹性
- 紧张悬疑 → 低沉配乐，暗色调，节奏紧凑
- 温馨治愈 → 柔和光线，慢节奏，暖色调

### Step 5: 色彩风格（Color Style）

问用户颜色风格，提供参考：

- 超写实波普风
- 低饱和度莫兰迪色系
- 高饱和度赛博朋克
- 温暖柔和的日式小清新

### Step 6: 时长与分镜数量

问用户视频时长（通常 15-30 秒），据此规划分镜数量。

分镜节奏参考：

| 时长 | 分镜数 | 每镜时长 |
|------|--------|----------|
| 15秒 | 5-6个 | 2-3秒/镜 |
| 30秒 | 8-10个 | 3秒/镜 |

### Step 7: 逐段生成时间线分镜

按「时间点 + 镜头规格 + 动作描述 + 音效」格式生成每个分镜：

分镜格式：

0:00 - 0:02：
镜头规格 +景别。角色动作 + 场景变化。
音效：[具体音效描述]

0:02 - 0:04：
...

镜头规格参考：
- 极近特写：50mm，适合开场钩子
- 中景：35mm，适合主场景
- 全景：28mm，适合展示全貌
- 推进：85mm，适合高潮/特写

## 最终输出格式

将以上内容组装成完整的提示词：

格式：[时长] / [机位风格] / [分镜数]个节拍 / [传播风格]

角色设定：
[角色描述]

风格：
[整体风格描述]

场景环境：
[场景描述]

氛围基调：
[氛围描述]

色彩风格：
[色彩方案]

时间线：
[逐段分镜]

## 使用时机

当用户提到以下任一关键词时使用：
- AI 视频提示词
- 视频分镜
- 即梦 / Jimeng
- Seedance / Kling / Sora
- AI 视频脚本
- 视频生成提示词
- "帮我写一个视频提示词"
- "生成视频分镜"

## 注意事项
- 每个问题都要给出参考示例，降低用户回答门槛
- 角色和场景描述要具体到「渔夫帽」「沙质水底」这种细节
- 时间线分镜是重点，要精确到秒，注明镜头规格
- 分镜之间的衔接要自然（切镜、推进、航拍等）
- 最后把完整提示词整理成可直接使用的格式
