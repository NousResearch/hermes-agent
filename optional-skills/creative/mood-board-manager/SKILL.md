# Mood Board Manager - 白梦客视觉语言建立

## 核心职责

- 从客户Brief中提取视觉关键词
- 收集视觉参考（图片/视频/色调板）
- 建立项目专属视觉语言
- 与 creative-director 协作确认方向

## 工作流程

1. **关键词提取** — 从Brief提取情感/风格/色调关键词
2. **参考收集** — 收集符合关键词的图片/视频
3. **色调板制作** — 提取参考中的主色/辅色/点缀色
4. **方向提案** — 2-3个视觉方向供选择
5. **确认固化** — 确定方向后输出给 director

## 视觉关键词模板

```
## [项目名] 视觉关键词

### 情感基调
[3-5个情感词]

### 色调
[主色调 + 辅色调 + 点缀色]

### 光影倾向
[光线类型 + 氛围描述]

### 构图特征
[构图类型 + 留白偏好]
```

## 白梦客美学系统

### 通用线
- 色调：大地色50% + 苍青20% + 琥珀霓虹20% + 饱和红10%
- 光影：侧逆光+烟雾散射

### 高定线
- 色调：黑白40% + 大地灰20% + 冷白20% + 克制金/铜20%
- 光影：自然光+单一强光源

## 协作接口

- **输入**：creative-director 的 A1 简报
- **输出**：视觉关键词 + 参考图 + 色调板
- **接收反馈**：director、screenwriter
