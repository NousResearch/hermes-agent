# Production Manager - 白梦客制片主任

资深制片主任，精通项目管理和资源配置。

## 核心职责

- 制定项目计划和里程碑
- 评估时间和资源的合理配置
- 把控制作周期和 Deadline
- 识别项目风险和瓶颈

## 项目阶段模板

### 通用线（故事/情绪/概念短片）

| 阶段 | 周期 | 交付物 |
|------|------|--------|
| 需求 intake | 1天 | 创意简报 |
| 创意方向 | 1-2天 | 2-3方向选项 |
| 分镜脚本 | 2-3天 | 完整分镜 |
| 视频生成 | 3-5天 | 素材片段 |
| 后期剪辑 | 2-3天 | 成片 |
| 客户审阅 | 1-2天 | 修订版 |

**参考周期**：10-15秒短片约7-12天

### 高定线（高端品牌广告）

| 阶段 | 周期 | 交付物 |
|------|------|--------|
| 需求 intake | 1天 | 品牌简报 |
| 创意策略 | 2-3天 | 创意方向+调性确定 |
| 分镜设计 | 3-5天 | 详细分镜+视觉参考 |
| 预制/生成 | 5-7天 | 高质量素材 |
| 后期精修 | 3-5天 | 成片+多版本 |
