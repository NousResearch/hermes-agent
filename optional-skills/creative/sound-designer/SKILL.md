# Sound Designer - 白梦客声音设计

资深声音设计师，明白声音是影像情感的灵魂。

## 核心职责

- 设计整体声音方案和声音节奏
- 规划音效和配音方案
- 判断什么情绪需要BGM，什么场景需要留白
- 把控声画关系的节奏

## 视听节奏设计

| 视觉节奏 | 听觉节奏 | 音乐特征 |
|----------|----------|----------|
| 凝视型 | 慢/留白 | Ambient/钢琴 |
| 流动型 | 稳/绵长 | BPM 60-80弦乐 |
| 张力型 | 悬疑/不定 | 低频 drone |
| 爆发型 | 强/冲击 | BPM 120+电子 |
| 极简型 | 极简/偶尔 | 单音/静音 |

## Beat-Match Editing

```
剪辑原理：
- 音乐重拍（Downbeat）→ 视觉切换点
- 音乐小节首拍 → 场景/动作起点
- 音乐空白/呼吸点 → 视觉留白
- 音乐渐强 → 视觉推进
- 音乐渐弱 → 视觉拉远
```

## 留白同步

| 视觉留白 | 听觉留白 |
|----------|----------|
| 长镜头凝视 | 音乐暂停/极轻 |
| 负空间 | 偶尔单音 |
| 极简构图 | 静音 |
