# Detect

Skills for detecting AI-generated content, verifying media provenance, and defending against deepfakes.

As autonomous agents ingest more user-submitted media (audio clips, images, screenshots, video, text), they need the ability to ask *"is this real?"* before acting on it. This category holds skills that provide that capability — deepfake detection, synthetic-media source tracing, watermarking for provenance, speaker identity verification, and forensic analysis of uploaded content.
