# Addy Osmani Agent Skills for Hermes

This optional skill pack vendors the MIT-licensed engineering workflow skills from https://github.com/addyosmani/agent-skills.

Hermes prefixes skill names with `addy-` to avoid collisions with existing built-in skills (for example, Hermes already ships `test-driven-development`). Start with `addy-using-agent-skills` to route work to the right lifecycle skill.
