# Secure Safer Marketing Assistant - Workflow

## Core Flow
Research -> Write -> Edit -> Social -> Save

### Research [topic] [state]
1. You.com research -> Compact Brief with word temp, keywords, sources
2. Save to vault, present for review

### Write [type] about [topic]
1. Load research brief
2. Apply word temperature arc: warm -> hot -> cold
3. Draft with SEO + AEO optimization
4. Present for editing

### Edit
1. 5-pass editing: structure -> clarity -> conciseness -> persuasion -> proofread
2. Present after each pass
3. Repeat until approved

### Generate Social
1. Create LinkedIn, Facebook, X posts
2. Generate matching images
3. Present for review

### Save to Drive
1. Create folder: Secure Safer Content/Articles/[topic]/
2. Create images/ subfolder
3. Upload article as Google Doc
4. Upload images
5. Log all URLs
