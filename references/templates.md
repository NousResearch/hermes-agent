# STATUS.md Templates

Reference these templates when creating or upgrading a project's status file.

## A) Full Template (Multi-day / Complex Projects)

```markdown
# 项目名称：[Name]
> 最后更新：[Date] | Git: [commit hash]

## 🎯 核心目标
[一句话描述项目终极目标]

## 📍 当前状态
[当前处于什么阶段，刚完成了什么]

## 📋 待办事项 (Todo)
- [x] 已完成 1
- [ ] 待办 2 (优先级高)
- [ ] 待办 3

## 🔑 关键上下文/决策
- 决策 1 (为什么这么做)
- 约束 1 (用户明确要求的事项)
- 链接 (相关 URL/文件路径)

## 🛑 暂停原因/遗留问题
[如果项目暂停，记录卡点或下次需要确认的问题]
```

## B) Light Template (Single Tasks < 3 Days)

```markdown
# 项目名称：[Name]
> 最后更新：[Date]

## 📍 当前状态
[一句话描述当前进度]

## 🔑 决策（可选）
[如果做了重要选择，记录原因。如没有可省略]

## 📋 下一步
- [ ] 接下来要做什么
```
