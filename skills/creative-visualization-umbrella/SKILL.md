# Creative Visualization Umbrella

Class-level skill for content visualization, combining infographics, article illustrations, and knowledge comics.

## Components

### Infographics (`baoyu-infographic`)
- Information density, 21 layouts × 21 styles.
- Data-rich guides, flowcharts, dashboards.

### Article Illustrations (`baoyu-article-illustrator`)
- Content-contextual illustrations.
- Type × Style × Palette consistency.

### Knowledge Comics (`baoyu-comic`)
- Narrative/biography/tutorial comics.
- Panel-by-panel storytelling, character consistency.

## Workflow
1. Analyze content and intent.
2. Select format (Infographic vs Illustration vs Comic) based on content structure.
3. Apply selected skill's methodology for prompts and generation.
4. Maintain consistency via style presets and reproducibility via stored prompts.
