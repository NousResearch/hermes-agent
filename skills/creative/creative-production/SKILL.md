# Creative Production

Use this skill for creative artifacts: designs, diagrams, mockups, generative graphics, animations, infographics, ASCII art/video, music/lyrics, and live visuals.

## Creative operating loop

1. Identify artifact type, audience, medium, dimensions, constraints, and delivery format.
2. Choose the simplest production path that can produce a real artifact.
3. Build a complete first version, then inspect it visually or render/play it.
4. Iterate on concrete critique: hierarchy, contrast, motion, pacing, readability, and brand/style fit.
5. Deliver the file/URL plus usage notes and verification evidence.

## Design and web mockups

- **Claude-style HTML artifacts / sketch variants**: create 2-3 real HTML options when comparing concepts; use one-off HTML for landing pages, decks, prototypes, and demos.
- **Popular web designs**: emulate known product design systems by extracting layout, spacing, typography, and interaction patterns—not logos or copyrighted assets.
- **DESIGN.md**: capture reusable design tokens and component guidance when a project needs a design system spec.
- **Pretext/p5.js**: use for creative browser demos, kinetic typography, generative art, shaders, interactivity, and WebGL/3D.

## Diagrams and explanatory visuals

- **Architecture diagrams**: dark SVG/HTML diagrams for systems, clouds, infra, and data flow.
- **Excalidraw**: hand-drawn architecture, flow, and sequence diagrams; save JSON and optionally upload for sharing.
- **Infographics / Baoyu-style**: choose layout and style family, structure content first, then render a dense visual summary.

## ASCII, animation, and video

- **ASCII art**: banners, cowsay/boxes, and image-to-ASCII.
- **ASCII video / Manim**: plan scenes, render short previews first, then export MP4/GIF; verify the output opens/plays.
- **TouchDesigner/ComfyUI**: check local runtime, models/nodes, and templates before running workflows; smoke-test a small graph.

## Music, audio, and text style

- **Songwriting/AI music**: write lyrics, structure, meter, style tags, and generation prompts.
- **Humanizer**: remove generic AI writing patterns while preserving meaning and the user's voice.

## Quality bar

Creative work must be real, inspectable, and polished enough to judge: no placeholder-only artifacts, no unrendered claims, and no fabricated previews.
