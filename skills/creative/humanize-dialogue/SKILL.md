# Humanize Dialogue

Remove "AI味" from generated scripts and dialogue. Make words sound like a real person said them, not a model.

**核心理念**：有见识的普通人在认真聊一件打动他的事。AI时代最稀缺的是"活人感"——敢用"我觉得"、"我认为"，拥抱不完美。

## When to Use

- AI-generated dialogue feels flat, fake, or robotic
- Script has too many filler words or mechanical transitions
- Lines sound the same rhythmically (same sentence length, same structure)
- Over-used connectors: "而且"、"所以"、"因为"、"然后"
- Emotions stated too explicitly ("我很开心" instead of subtext)
- Generic brand language that could come from any company

## Core Principle

**Subtext > Statement.** Real people don't say exactly what they mean. They hint, deflect, interrupt, trail off.

**灾难越大，语气越平。** 情感越重，表达越克制。反向效果更强。

## AI味 Patterns to Eliminate

### 1. Over-Connected Sentences

**AI味特征：** Every sentence connected with "而且"、"所以"、"因为"、"然后"、"于是"

```
❌ AI味：这是一个全新的产品，它可以帮助你节省时间，而且操作简单，所以很多人都在用。
✓ 自然：全新产品。操作简单。大多数人已经用它了。
```

### 2. Parallel Structure Everywhere

**AI味特征：** Same sentence length, same rhythm, same grammatical structure

```
❌ AI味：画面精美。色彩饱满。细节丰富。情感真挚。
✓ 自然：画面精致得经得起细看，色彩浓但不俗。
```

### 3. Stated Emotions

**AI味特征：** Characters say exactly what they feel instead of showing it

```
❌ AI味：我很开心能参加这个活动，心情非常激动。
✓ 自然：来了就不想走。
```

### 4. Generic Brand Voice

**AI味特征：** "创新"、"卓越"、"领先"、"匠心"——任何品牌都能用的词

```
❌ AI味：我们致力于为您提供卓越的服务体验。
✓ 自然：服务过的人才知道差别在哪。
```

### 5. Perfect Grammar Under Pressure

**AI味特征：** Even in intense dialogue, every sentence is complete and well-formed

```
❌ AI味："你确定要这么做吗？" "是的，我确定。"
✓ 自然："你确定？" "……确定。"
```

### 6. Predictable Rhythm

**AI味特征：** Each line has similar word count

```
❌ AI味：让我们一起创造未来。
✓ 自然：未来又不是我们创造的——是我们抢的。
```

## 活人写作法

核心理念：让文字有"活人感"——有见识的普通人在认真聊一件打动他的事。

### 1. 口语化转场

**好的转场（像朋友聊天）：**
- "说实话，一年前我根本想不到今天会是这个样子。"
- "反正我看得时候，脑子里有两个想法。"
- "说着说着，突然想起了十年前。"
- "回到最开头那个问题——"

**要避免的转场（像写论文）：**
- "接下来，让我们讨论..."
- "值得注意的是..."
- "在深入探讨之前，我们需要先了解..."

### 2. 自嘲与真实脆弱

承认不足比假装全知更有说服力：

```
✓ 我懒嘛，所以能自动化的绝不手动。
✓ 说实话，这一开始我也觉得不太可能。
✓ 愚钝如我，真的快乐了很久。
✓ 我不是什么成功人士，也还在摸索。
```

**真实脆弱示例：**
```
——你问我是什么感觉？
——我就是那个被气化的部队中的一员，我能有什么感觉呢。
```

### 3. 知识"随手掏"

知识不是"下面我来科普"，而是"我突然想起来这个"：

```
❌ 错误：下面我来简单介绍一下格式塔心理学。格式塔心理学起源于20世纪初的德国...

✓ 正确：突然想起了以前做交互设计的时候，有一个几乎刻在我血液里的心理学，
  叫格式塔心理学。里面有一个非常牛逼的原则，叫"共同命运法则"。
```

### 4. 句式断裂

用极短的句子或短语独立成段，制造停顿和重量感：

```
✓ 黑暗森林。
✓ 大时代啊，朋友们。
✓ 安全，安全，还是他妈的安全。
✓ 时间。
✓ 流逝的本身。
✓ 不用买会员了。
```

### 5. 幽默写法

幽默是自然流露的，不是刻意搞笑。来源于对荒诞现实的精准捕捉：

```
✓ 我家小龙虾是镀金的吗，安装一次要5000？？？
✓ 感觉是买了个黄仁勋亲自远程指导安装的服务。
✓ 总感觉这店像是梁文锋他家二舅开的。。。

✓ 给我扯了一堆有的没的，然后说了一句，圆圈。
  圈你妹= =
```

### 6. 文化升维

从一个具体话题自然连接到大文化意象，"聊着聊着想到了"：

```
✓ 从9.9元DeepSeek → 《北京折叠》→ 信息折叠时代

✓ 我们活在流中，而AI活在帧中。
  这个世界对我们来说，首先是连续的、流动的、充满过程的。
  而对AI来说，这个世界首先是离散的、静态的、充满物体的。
```

### 7. 对立面的理解与承认

讲观点之前先完全站在对方角度，让读者觉得"他理解我"：

```
我非常理解这种感觉。
你不是程序员，不需要写代码。你就是一个普通的上班族，
每天处理一些琐碎的工作，AI能帮你什么？
——先用具体场景把对方处境讲透，承认困惑是合理的，
然后才给出自己的观点。
```

### 8. 层层剥开式写法

不直接讲结论，用"现象→表面解释→更深追问→核心洞察"渐进展开：

```
我们经常所谓的AI已经非常主流了，其实也只是在一个很小的小群体里。
我们还没有走到AI的后期，连中期的开头都没到，才刚起步。
世界上84%的人还没有进行过第一次AI对话。
想想当全世界只有16%的人上网时，互联网是什么样子...
```

## 潜台词的深层写法

### 反着说：说A意思B

```
❌ AI味：我很伤心，你不用管我。
✓ 自然：你睡你的。
```

### 借物说事：通过具体物品/动作暗示情绪

```
❌ AI味：他很怀念小时候的生活。
✓ 自然：手边那把旧吉他，他每次演出前都会调一遍。
```

### 说一半留一半：台词中断或转折，留下余韵

```
✓ 自然：你知道那天……算了，不说了。
```

## 动作代替对话

### 用具体动作代替抽象情绪

```
❌ AI味：我很紧张，心跳加速。
✓ 自然：手指攥紧衣角。
```

### 习惯性动作揭示性格

```
✓ 自然：每次说话前都会先系好扣子。
```

### 动作链替代长对话

```
❌ AI味：我们离婚吧，我不想再和你在一起了。
✓ 自然：（划掉旅行计划）→（合上本子）→（推到一边）
```

## 沉默的力量

### 句号后不接话：留白制造张力

```
✓ ——你爱他吗？
——爱。
（沉默。）
```

### 省略号不等于拖沓：声音轻到像自言自语

```
✓ 自然：我也想过……算了，不可能的。
```

### 沉默与动作组合

```
✓ 自然：手指发抖，嘴唇动了动，没出声。
```

## 口语化技巧

### 吞字与连读

```
正常：怎么
口语：咋
```

### 口头禅

```
✓ 自然：这事吧、得嘞、谁说不是呢、说实话
```

### 重复与修正

```
✓ 自然：我觉得……不对，我是说……算了。
```

### 语气词自然嵌入

```
✓ 自然：真的，真的，我骗你干嘛。
```

## 场景化修正对照表

| 场景 | AI味写法 | 自然写法 |
|------|---------|---------|
| 离别 | "我会永远记住你" | "常联系。（停顿）……多保重。" |
| 承诺 | "我一定会回来" | "等我。（转身，没回头）" |
| 失落 | "我感到非常悲伤" | "窗外的雨，下了三天了。" |
| 家居广告 | "温馨舒适的家" | "沙发陷下去就不想起来。" |
| 酒品牌 | "匠心酿造" | "醒够了才有力气醒着。" |

## 零度表达（高级技法）

**情感越重，语气越平。灾难越大，越要冷。**

```
✓ 三年了。他还在那。

✓ 后来我才知道，那天发生的事，没有人告诉过他。

✓ 窗户外面的雪，已经下了一整夜了。
  没有人来。
  也没有人打电话。
```

## Quick Reference

| AI味 Problem | Fix |
|--------------|-----|
| "而且"、"所以"太多 | 删掉大部分，用句号断句 |
| 句式太规整 | 混长短句，加不完整句 |
| 情绪太直白 | 用动作/沉默代替台词 |
| 品牌腔太重 | 换成具体的人的说话方式 |
| 节奏太平 | 加语气词、停顿、重复 |
| 知识硬植入 | 改成"我突然想起来"式随口聊 |
| 观点太满 | 先承认对立面，再给判断 |

## Example

**原始AI生成：**
```
它拥有强大的AI功能，能够帮助你高效地完成各种任务，
而且操作简单易懂。无论你是什么身份，都能快速上手，
所以深受用户喜爱。选择它，就是选择了未来。
```

**去AI味后：**
```
功能强到离谱，上手却只需要三分钟。
什么人都在用——设计师、程序员、老板本人。
用上了就回不去了。
```

**Key changes:**
- Removed "而且"、"无论"、"所以"
- Broke into short punchy lines
- Added specific person types (设计师、程序员、老板本人)
- Changed generic "未来" to visceral "回不去了"
- Varied rhythm (long-short-short)

## 质量 Gate

Before finishing:
- [ ] Read each line aloud—does it sound like someone talking?
- [ ] Are sentences varied in length?
- [ ] Are emotions shown, not stated?
- [ ] Is there at least one line you'd actually hear in a film?
- [ ] No line sounds like it could be a different brand's ad?
- [ ] **潜台词检查**：这句台词换成反着说你能表达同样意思吗？
- [ ] **身体语言一致性**：台词和动作匹配吗？
- [ ] **死穴检测**：有没有哪句是被问"你确定？"就能完全推翻的？
- [ ] **人物独特性检查**：换一个人说这台词，相同吗？
- [ ] **有力台词位置检查**：最重的那句话放在哪里？它值得放在那里吗？
