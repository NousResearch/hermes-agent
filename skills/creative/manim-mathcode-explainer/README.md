# manim-mathcode-explainer

ELI5 animated math explainer videos grounded in Lean 4 formal proofs.
Produces ~2–3 minute Manim videos with per-scene narration, built from
a verified proof generated by MathCode.

## Prerequisites

**MathCode**
```bash
git clone https://github.com/math-ai-org/mathcode.git ~/mathcode
cd ~/mathcode && bash setup.sh
echo 'ANTHROPIC_API_KEY=your_key_here' >> ~/mathcode/.env
```

**Manim**
```bash
pip install manim
```

**ffmpeg**
```bash
brew install ffmpeg      # macOS
sudo apt install ffmpeg  # Linux
```

**LaTeX: do not install.** This pipeline uses Cairo (`Text()`) everywhere. `MathTex` is not used and will crash without a full LaTeX install.

## Verify setup
```bash
bash skills/creative/manim-mathcode-explainer/scripts/setup.sh
```

## Usage
> "Make an ELI5 explainer video for [theorem]. Use the manim-mathcode-explainer skill."

## Examples
- Jordan's Lemma — mirrors, spins, Grover's algorithm
- No-Cloning Theorem — spinning coins, quantum encryption
