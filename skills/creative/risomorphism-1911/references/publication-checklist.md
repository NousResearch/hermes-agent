# ASCII Art Pipeline Publication Checklist

## Public package
- clear README with images first
- named presets documented with exact commands
- at least 3 flagship examples ready
- gallery page works locally
- diagnostics command demonstrated

## Hermes skill
- `SKILL.md` has valid frontmatter
- references and scripts live under the skill directory
- docs generated with `website/scripts/generate-skill-docs.py`
- skill appears in bundled skills docs/catalog

## Quality gate
- examples are recognizable at a glance
- density reads as deliberate, not garbled
- side-by-side preset comparison exists
- at least one animated eikon example is included
