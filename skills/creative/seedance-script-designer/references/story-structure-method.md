# Story Structure Method

Use this reference before converting a script into Seedance 2.0 visual assets and shot prompts.

This method is inspired by screenwriting structure principles: do not copy source text; apply the ideas as practical production checks.

## 1. Dramatic Core

Before drawing characters or splitting shots, identify:

- Protagonist: whose desire drives the scene or story.
- Object of desire: what the protagonist wants externally.
- Inner need: what emotional or moral change may be required.
- Opposition: person, system, environment, inner flaw, time pressure, secret, or impossible choice.
- Stakes: what is lost if the desire fails.
- Value polarity: the emotional or dramatic value that moves, such as safety/danger, trust/betrayal, hope/despair, love/loneliness, freedom/captivity, victory/defeat.

If the script lacks a clear desire or opposition, ask one concise question or state a conservative assumption.

## 2. Scene Test

Each scene should be a small story unit, not only a location.

For every scene, answer:

- What does the character want right now?
- What blocks them?
- What action do they take?
- What unexpected result creates a gap between expectation and reality?
- What value changes from beginning to end?
- What is the turning point?

If no value changes, the scene is likely exposition or atmosphere. Compress it, merge it into another scene, or make it a visual montage unless the user specifically wants mood-only material.

### Compression Operations

Use one of these concrete operations when a scene lacks value change or repeats information:

- Merge: fold the useful line, prop reveal, or emotional reaction into the previous or next scene that already has a stronger value shift.
- Montage: convert repeated exposition into 2-4 visual nodes with clear progression, such as location -> object -> reaction -> consequence.
- Transition shot: keep only one atmospheric image, prop close-up, movement cue, or sound bridge when the scene mainly connects locations or time.
- Delete: remove the scene when it adds no new information, emotion, relationship change, visual motif, or rhythm.
- Elevate: if the scene must stay, add a choice, obstacle, reveal, reversal, or irreversible action so it earns its screen time.

When compressing, preserve the information that changes future behavior. Drop explanatory lines that only restate what the audience already knows.

## 3. Beat Breakdown

Break scenes into beats when character action or reaction changes.

Beat categories:

- Intent beat: character chooses a tactic.
- Pressure beat: obstacle pushes back.
- Discovery beat: new information changes the situation.
- Reversal beat: expected result flips.
- Decision beat: character chooses under pressure.
- Consequence beat: result lands emotionally or physically.

A shot can cover one beat or several simple beats. Use a new shot when the beat changes emotion, speaker, location, tactic, or visual focus.

## 4. Turning Points and Progression

Design shot order around escalation.

- Inciting event: the first disturbance that makes normal life impossible.
- Progressive complication: each attempt creates a harder problem.
- Crisis: a pressured choice between meaningful alternatives.
- Climax: the irreversible action or revelation.
- Resolution: the emotional or practical result.

For short Seedance clips, these may be compressed into 3-5 beats:

1. setup;
2. disturbance;
3. escalation;
4. turning point;
5. payoff or unresolved hook.

## 5. Character Design Implications

Character assets should reflect dramatic function, not only appearance.

- Desire should influence posture, gaze, movement rhythm, and prop choices.
- Inner contradiction should appear in costume contrast, gesture, or facial tension.
- Opposition should have a visual force: blocking, shadow, crowd, barrier, distance, weather, architecture, or another character.
- Transformation should have a visible before/after state: expression, costume damage, lighting shift, camera distance, or color temperature.

## 6. Scene and Camera Implications

Translate dramatic structure into camera decisions:

- Desire: forward movement, close framing, tracking, eye-line clarity.
- Resistance: obstacles in foreground, blocked composition, handheld instability, tighter space.
- Discovery: rack focus, reveal pan, insert shot, slow push-in.
- Reversal: cut, whip pan, sudden silence, change of angle, lighting shift.
- Crisis: close-up, held shot, reduced movement, audible breath or silence.
- Climax: decisive action, strong camera move, impact sound, clear visual result.
- Resolution: wider frame, release of tension, changed color/light/music.

## 7. Output Rules

Add a story-structure layer before final Seedance prompts:

- Logline or scene premise.
- Character desire and opposition.
- Value shift per scene.
- Beat table.
- Turning point and payoff.
- Notes on weak or decorative scenes.

Then use this structure to decide which assets, expressions, props, scenes, music cues, and camera moves are actually necessary.
