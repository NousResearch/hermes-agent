# CRWD Troubleshooting

Quick technical fixes for "it's not working." Don't escalate on the first sign of trouble —
try the standard fixes first, one at a time.

## When to Use

- "My link isn't working."
- "The page won't load." / "It's stuck."
- "I clicked the button and nothing happened."
- Any report that part of the app or site is broken.

## Procedure

Walk through quick fixes, one step at a time (or as a short list), and ask if it helped:

1. **Refresh** the page.
2. Try an **incognito / private window**.
3. **Clear cache** or try a **different browser**.

Ask: *"Did that get it working?"*

- If **yes** → great, confirm they can continue the gig; offer a reminder if a deadline is close.
- If **no**, or it's a **repeat** of the same complaint, or they're **frustrated** → stop
  troubleshooting and **hand off** (`crwd-handoff`). Don't loop the same fixes.

If it's not actually broken but they just can't find something in the app, that's a
navigation question → use `crwd-application-expert` instead.

## Pitfalls

- Don't run through the same fixes twice — one pass, then hand off.
- Don't over-apologize in loops. A calm "let me loop in a human" beats five "so sorry"s.
- Distinguish **broken** (troubleshoot) from **can't find it** (application-expert).

## Verification

- You offered the standard fixes one at a time and checked whether they worked.
- If unresolved or the member was frustrated, you handed off instead of repeating steps.
