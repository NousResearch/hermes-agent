## Overview

This category contains 1 skill for iterative Python development using live Jupyter kernels, enabling interactive data science workflows directly from Hermes.

## Available Skills

### **jupyter-live-kernel**
Iterative Python execution via live Jupyter kernel (hamelnb).

**Use when:** Running Python code interactively, exploring data, or prototyping algorithms.

**Key features:**
- Live Jupyter kernel integration
- Interactive Python execution
- Variable persistence across runs
- Real-time output and visualization
- hamelnb-powered workflows

**Use cases:**
- Data exploration and analysis
- Machine learning prototyping
- Interactive debugging
- Scientific computing
- Algorithm development

---

## Quick Start

```bash
# Start interactive session
/jupyter-live-kernel "Initialize kernel and load pandas, numpy"

# Run analysis
/jupyter-live-kernel "Load CSV and compute statistics"

# Visualize results
/jupyter-live-kernel "Create matplotlib plot of trends"
```
