# DevOps Skills

Event-driven automation via webhook subscriptions.

## Overview

This category contains 1 skill for webhook-based automation, enabling event-driven agent workflows triggered by external events.

## Available Skills

### **webhook-subscriptions**
Webhook subscriptions for event-driven agent runs.

**Use when:** Automating responses to external events like deployments, alerts, or system notifications.

**Key features:**
- Webhook endpoint creation
- Event-driven automation
- Subscription management
- Event filtering and routing
- Automated agent triggers

**Use cases:**
- CI/CD pipeline integration
- Deployment notifications
- System alert responses
- Third-party service integrations
- Automated incident response
