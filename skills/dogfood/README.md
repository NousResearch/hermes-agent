# Dogfood Skills

## Overview

This category contains 1 skill for exploratory quality assurance of web applications, automating bug discovery and evidence collection.

## Available Skills

### **dogfood**
Exploratory QA of web apps: find bugs, collect evidence, generate reports.

**Use when:** Testing web applications, discovering bugs, or conducting QA on internal tools.

**Key features:**
- Automated exploratory testing
- Bug discovery and documentation
- Evidence collection (screenshots, logs)
- QA report generation
- Web app analysis

**Use cases:**
- Pre-release QA testing
- Internal tool validation
- Bug hunting and discovery
- Automated regression testing
- User journey testing
