# Email Skills

Terminal-based email management via IMAP/SMTP.

## Overview

This category contains 1 skill for email management from the command line using the Himalaya CLI.

## Available Skills

### **himalaya**
Himalaya CLI: IMAP/SMTP email from terminal.

**Use when:** Managing email from command line, automating email workflows, or building email-based systems.

**Key features:**
- IMAP email reading
- SMTP email sending
- Terminal-based interface
- Multiple account support
- Email filtering and search
- Attachment handling

**Use cases:**
- Command-line email management
- Automated email responses
- Email monitoring and alerts
- Scripted email workflows
- Server-side email access

---

## Quick Start

```bash
# Read emails
/himalaya "Show unread emails from today"

# Send email
/himalaya "Send email to team@company.com: Deployment complete"

# Search
/himalaya "Search emails for 'invoice' in last 30 days"
```
