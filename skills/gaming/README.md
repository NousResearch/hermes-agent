# Gaming Skills

Game server hosting and automated gameplay for Hermes Agent.

## Overview

This category contains 2 skills for gaming-related workflows — hosting modded Minecraft servers and automated Pokemon gameplay via headless emulation. Whether you're managing game servers for communities or building AI-powered game bots, these skills provide gaming infrastructure and automation.

## Available Skills

### Server Hosting

#### **minecraft-modpack-server**
Host modded Minecraft servers with CurseForge and Modrinth modpack support.

**Use when:** Running Minecraft servers with custom modpacks for communities or testing.

**Key features:**
- CurseForge and Modrinth modpack support
- Automated server setup
- Modpack version management
- Server administration tools

---

### Automated Gameplay

#### **pokemon-player**
Play Pokemon games via headless emulator with RAM reads and automated controls.

**Use when:** Building Pokemon game bots or AI training environments.

**Key features:**
- Headless emulator support
- RAM state reading
- Automated game controls
- Save state management

---

## Quick Start

```bash
# Minecraft server
/minecraft-modpack-server "Create server with All The Mods 9"

# Pokemon bot
/pokemon-player "Auto-battle wild Pokemon to level 20"
```
