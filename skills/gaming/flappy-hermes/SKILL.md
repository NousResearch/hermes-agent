# Flappy Hermes

A Flappy Bird-style browser game created entirely by Hermes Agent (Nous Research) during a live conversation session.

## How to Play

1. Open `flappy-hermes.html` in any modern browser
2. Press SPACE or click to make the bird fly
3. Avoid the pipes and survive as long as possible
4. Beat your high score!

## What Hermes Did

This game was built step by step through natural language conversation:

- Generated the full HTML5 + Canvas game from scratch
- Tuned gravity, jump force, pipe speed and gap based on user feedback
- Deployed the file via SSH to a remote WSL2 server (Ubuntu 24.04)
- Copied to Windows Desktop automatically

## Game Features

- Smooth canvas-based animation
- Animated bird with wing flap and rotation
- Starry night sky background
- Neon purple Hermes-themed UI
- Real-time score and high score tracking
- Adjustable difficulty settings in source

## Tech Stack

- Pure HTML5 + JavaScript (no dependencies)
- Canvas 2D API for rendering
- Works in any modern browser
