# BeSoul — No Talking Heads Storytelling Variant

Use when user rejects ad look and explicitly asks to avoid women talking on camera.

## Goal
Preserve 60s viral storytelling mechanics while removing presenter footage.

## Non-negotiables
- No female talking heads.
- No direct-to-camera monologue shots.
- No UGC selfie style.

## Allowed visual palette
- Abstract cosmic/mystic textures.
- App screen closeups / UI overlays.
- Symbolic props (chart wheels, moon motifs, notebooks, hands-only interactions).
- Silhouettes and environmental mood shots.

## Rhythm
- Hook in first 1–2s via pattern interrupt visual + line.
- Micro visual turns every 4–7s.
- Save-trigger closure in final 8–12s.

## Verification checklist (before delivery)
Sample frames at ~10%, ~45%, ~80% timeline:
1) Any woman talking directly to camera? If yes → reject and re-cut.
2) Does it read as ad montage instead of intimate story? If yes → reduce promo UI density and increase symbolic b-roll.
3) Is CTA soft (save/reflect) rather than hard sell? Must be soft.
