# BeSoul Viral Curiosity 60s Template (EN)

## Objective
Create a 55–60s vertical reel optimized for saves/shares using curiosity-storytelling (not hard sell).

## VO skeleton (fill-in)
- Hook (0:00–0:06): "<pattern interrupt line>"
- Friction (0:06–0:14): "<why generic / repeated loop felt unresolved>"
- Reveal (0:14–0:24): "<AI natal chart insight mechanism>"
- Relatable insights (0:24–0:36): "<2 concrete personal realizations>"
- Honesty line (0:36–0:47): "not magic, not therapy, but a mirror"
- CTA (0:47–0:60): "save this, run your chart tonight, see what resonates"

## Shot map template (12 shots)
1. 0:00–0:04 hook visual
2. 0:04–0:08 friction visual
3. 0:08–0:12 app reveal
4. 0:12–0:16 chart details
5. 0:16–0:20 pattern symbol visual
6. 0:20–0:24 reflective pause
7. 0:24–0:29 insight card 1
8. 0:29–0:34 insight card 2
9. 0:34–0:39 honesty beat
10. 0:39–0:45 calm clarity beat
11. 0:45–0:51 CTA prep
12. 0:51–end final save prompt

## Mix guideline
- VO: primary
- BGM: ~30% under VO
- No subtitles unless requested
