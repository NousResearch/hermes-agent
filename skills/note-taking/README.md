# Note-Taking Skills

Obsidian vault integration for knowledge management.

## Overview

This category contains 1 skill for reading, searching, and creating notes in Obsidian vaults.

## Available Skills

### **obsidian**
Read, search, and create notes in the Obsidian vault.

**Use when:** Managing knowledge in Obsidian, automating note creation, or building workflows around your Obsidian vault.

**Key features:**
- Read notes from vault
- Search across all notes
- Create new notes
- Update existing notes
- Tag and link management
- Vault structure navigation

**Use cases:**
- Automated note creation
- Knowledge base queries
- Daily note automation
- Research organization
- Zettelkasten workflows

---

## Quick Start

```bash
# Create note
/obsidian "Create daily note with template"

# Search vault
/obsidian "Search for notes about 'project planning'"

# Update note
/obsidian "Add task to 'Today' note: Review PR #123"
```
