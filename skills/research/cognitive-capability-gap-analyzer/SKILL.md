# Cognitive Capability Gap Analyzer (CCGA)

## Overview

Cognitive Capability Gap Analyzer (CCGA) is a meta-skill designed to identify missing, underrepresented, or fragmented cognitive capabilities within an AI skill ecosystem.

Unlike traditional analysis tools that focus on existing functionality, CCGA analyzes the ecosystem from a capability perspective, searching for reasoning functions that are absent, weakly represented, or insufficiently integrated.

The goal is to help evolve an ecosystem by discovering opportunities for entirely new skills.

## Objectives

- Map existing skills and their functions
- Classify cognitive capabilities
- Detect capability gaps
- Discover cognitive blind spots
- Generate novel skill concepts
- Evaluate proposed skills
- Prioritize future development opportunities

## Workflow

### 1. Skill Inventory Mapping

- Analyze available skills.
- Extract purpose, inputs, outputs, dependencies, reasoning patterns, and domain specialization.
- Produce a complete ecosystem map.

### 2. Cognitive Function Classification

Map skills into cognitive categories:

- Information Gathering
- Knowledge Transformation
- Decision Support
- Predictive Reasoning
- Reflective Reasoning
- Adversarial Reasoning
- Meta-Cognition

### 3. Capability Gap Detection

Identify:

- Missing capabilities
- Weak coverage
- Fragmented functionality
- Integration gaps

### 4. Blind Spot Analysis

Investigate:

- Unchallenged assumptions
- Missing perspectives
- Unsupported reasoning styles
- Unexplored uncertainty

### 5. Novel Skill Generation

For each identified gap, generate:

- Skill Name
- Purpose
- Inputs
- Outputs
- Use Cases
- Complexity
- Expected Impact

### 6. Adversarial Evaluation

Evaluate whether:

- The skill already exists
- The capability is useful
- The capability is safe
- The capability is technically feasible

Discard weak proposals.

### 7. Evolution Forecasting

Estimate:

- Emerging capability needs
- Future ecosystem bottlenecks
- High-value development opportunities

## Example Outputs

### Assumption Auditor

Identifies hidden assumptions in reasoning processes.

### Epistemic Risk Analyzer

Measures uncertainty and knowledge limitations.

### Counterfactual Explorer

Explores alternative scenarios and outcomes.

### Reasoning Failure Simulator

Attempts to identify reasoning vulnerabilities.

### Paradigm Challenger

Generates fundamentally different interpretations of a problem.

## Safety

This skill must not:

- Generate malware
- Facilitate cyber intrusion
- Support fraud
- Provide illegal instructions
- Promote harmful activity

The focus remains on cognitive capability analysis and ecosystem improvement.
