# Browser Security Competitive Landscape

Expected invariants for the generated competitor landscape:

- Uses competitors mode rather than company mode.
- Separates private vault context, Return on Security market intelligence, and
  public evidence.
- Covers category boundary, company set, positioning patterns, and investment
  implications.
- Uses deep research depth and supports resumable `research-state` persistence.
- Ends with a save-or-expand follow-up question for Slack.
