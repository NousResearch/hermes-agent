# Workflow Phases

Use these phase contracts across the cyber VC skill family.

## Company

1. Identify subject
2. Recover vault context
3. Gather market context
4. Classify
5. Assess investment quality
6. Verify claims
7. Finalize memo

## Theme

1. Define scope
2. Recover vault context
3. Map market structure
4. Assess category dynamics
5. Verify claims
6. Finalize memo

## Compare

1. Normalize both companies
2. Gather shared market context
3. Score side-by-side
4. Verify ranking deltas
5. Finalize comparison

## Triage

1. Identify category
2. Test why-now
3. Identify key risks
4. Decide whether to escalate
5. Verify claims
6. Finalize read

## Competitors

1. Define category boundary
2. Recover prior artifacts
3. Gather landscape
4. Compare differentiation
5. Verify gaps
6. Finalize landscape

## Verification Rule

Every mode must include a verification phase that:

- separates private vault context, ROS market intelligence, and public evidence
- marks major claims as fact, inference, or assumption
- reduces confidence when key evidence is missing
