# Cyber VC Analyst v1.1.0

Date: 2026-07-02

## What changed

- Documented the Slack-first workflow for `!cyber-vc-analyst` thread usage.
- Added a private-repo blueprint so the Hermes skill can sit in front of a
  broader private cyber investment workflow repository.
- Added starter schemas for company analysis, theme analysis, and evidence
  ledger structures.
- Added starter fixtures for Red Access company analysis and the
  SOC Automation / AI SOC thematic run.
- Added compare and triage scaffolding so the structured Slack modes have
  matching schemas and fixtures.
- Tightened Slack-first output prompts so company, theme, compare, and triage
  replies end with explicit save-to-vault or expand-next follow-up questions.
- Kept company output behavior unchanged.
- Kept the thematic output default at `3.Areas/cyber futures frontier`.

## Why this release exists

This release turns the skill from a single prompt bundle into a more durable
workflow surface that can be productized in Slack and synchronized with a
private research repository.

## Known limits

- The Hermes runtime skill is still the execution surface, not the canonical
  prompt-authoring repository.
- Fixtures currently capture required invariants rather than full golden memo
  bodies.
- Slack workflow guidance is documented here, but no dedicated Slack plugin or
  custom UI surface has been added.
