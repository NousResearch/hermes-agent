# Research Methodology Guide

## Research Strategies by Question Type

### Technology Evaluation
Structure: Performance → Ecosystem → Adoption → Migration → Risk
Search queries:
1. "[tech] performance benchmarks 2026"
2. "[tech] vs [alt] comparison production"
3. "[tech] ecosystem libraries tools"
4. "[tech] adoption case studies companies"
5. "[tech] limitations challenges problems"
6. "[tech] migration from [current_stack]"

### Market Analysis
Structure: Size → Growth → Players → Trends → Regulation
Search queries:
1. "[market] size revenue 2026"
2. "[market] growth rate CAGR forecast"
3. "[market] key players market share"
4. "[market] trends predictions future"
5. "[market] regulation policy compliance"

### Competitive Analysis
Structure: Features → Pricing → Market Share → Strengths → Weaknesses
Search queries:
1. "[product] features capabilities review"
2. "[product] vs [competitor] comparison"
3. "[product] pricing plans enterprise"
4. "[product] customer reviews pros cons"
5. "[product] market share adoption"

### Academic Review
Structure: Background → Methods → Results → Limitations → Future
Search queries:
1. "[topic] survey paper 2025 2026"
2. "[topic] recent advances arxiv"
3. "[topic] benchmark results SOTA"
4. "[topic] limitations open problems"
5. "[topic] future directions research"

## Source Evaluation Checklist

For each source, evaluate:
- Author credentials and expertise
- Publication date (prefer < 1 year)
- Evidence provided (data, citations, benchmarks)
- Potential bias (vendor blog vs independent analysis)
- Corroboration (do other sources agree?)

## Cross-Validation Protocol

1. Find 3+ sources for each key claim
2. If all agree → HIGH confidence
3. If 2/3 agree → MEDIUM confidence, note dissent
4. If sources conflict → LOW confidence, present both sides
5. If only 1 source → Tag as UNVERIFIED

## Common Research Pitfalls

| Pitfall | How to Avoid |
|---|---|
| Confirmation bias | Actively search for opposing views |
| Recency bias | Include historical context |
| Authority bias | Verify claims independently |
| Survivorship bias | Look for failed examples too |
| Selection bias | Use diverse source types |
