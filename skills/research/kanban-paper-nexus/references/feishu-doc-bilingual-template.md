# 飞书文档双语模板（T5）

**1 论文 = 1 doc**（registry canonical id）。同论文仅 append。

**叙事顺序：** 核心总结 → CEL 表 → 阅读地图 → 方法 → 对标 → 实验审计 → **参考方向** → 工具索引。

## 写作规则

1. 一级节：**中文 block** 在前，*English* 在后（1 段或 3 bullet，不扩写废话）
2. 数字格式：`指标 @ 设定`（例：`MATH pass@1 51.7%, 7B, no tool`）
3. 区分：**作者声称** vs **我们认为**（审计结论用后者）
4. 【待填】必须在 T5 全部替换掉 — QA 会 FAIL

## 核心总结（必填）

**中文 5 条：** 论点 | 方法 | 结果（带数） | 局限 | 参考方向（复现/产品/对标各 1 条）

**English：** 与上逐条 mirror，每条 ≤2 行。

## CEL 表（来自 T1，勿缩水）

| ID | 主张 | 证据 | 强度 | 局限 |

## 参考方向（必填，可执行）

**中文：**
- **复现：** 章节 + 资源（GPU、数据、代码链接）
- **产品化：** 场景 + 必建护栏（评测/安全/成本）
- **对标：** 2–3 篇论文或 repo + 一句话差异

**English:** mirror.

## IM（编排一条）

```
📄 论文精读 · {canonical_id}
【论点】{thesis_one_liner}
【结论】{一条最强数字或发现}
【文档】{新建|追加} {url}
【任务】{task_id 列表}
```

不要复制全文进 IM；细节在 doc。
