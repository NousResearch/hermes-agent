# 飞书文档双语模板（T5）

**1 论文 = 1 doc**（registry canonical id）。同论文仅 append。

**飞书云文档显示名（在线文档名）：** `[{canonical_id}] {title_zh}` — 必须用 T0 handoff 的中文题名，禁止用英文 arXiv 标题。

**叙事顺序：** 核心总结 → CEL 表 → 阅读地图 → 方法 → **核心公式** → **运行时策略细节** → 对标 → 实验审计 → **边界分析** → **软硬件 Delta** → **参考方向** → 工具索引。

## 写作规则

1. 一级节：**中文 block** 在前，*English* 在后（1 段或 3 bullet，不扩写废话）
2. 数字格式：`指标 @ 设定`（例：`MATH pass@1 51.7%, 7B, no tool`）
3. 区分：**作者声称** vs **我们认为**（审计结论用后者）
4. 【待填】必须在 T5 全部替换掉 — QA 会 FAIL
5. 创建/追加飞书文档时，命令必须写成 `lark-cli docs +create --api-version v2 ...` / `lark-cli docs +update --api-version v2 ...`

## 核心总结（必填）

**English：** 与上逐条 mirror，每条 ≤2 行。

## 核心公式（必填，若论文含目标函数/损失/策略）

**中文：**
