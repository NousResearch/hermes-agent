# Smart Home Skills

Philips Hue light control and automation.

## Overview

This category contains 1 skill for controlling Philips Hue lights, scenes, and rooms via the OpenHue CLI.

## Available Skills

### **openhue**
Control Philips Hue lights, scenes, rooms via OpenHue CLI.

**Use when:** Automating home lighting, creating scenes, or integrating lights into workflows.

**Key features:**
- Individual light control
- Scene management
- Room-based control
- Brightness and color adjustment
- Schedule automation
- CLI-based integration

**Use cases:**
- Automated lighting schedules
- Workflow-triggered lighting
- Focus mode lighting
- Notification lights
- Smart home integration

---

## Quick Start

```bash
# Control lights
/openhue "Turn on office lights to 80% brightness"

# Set scene
/openhue "Activate 'Focus' scene in office"

# Automation
/openhue "Dim all lights to 30% at 10pm"
```
