# Social Media Skills

X/Twitter automation and management.

## Overview

This category contains 1 skill for X/Twitter interaction via the xurl CLI using the v2 API.

## Available Skills

### **xurl**
X/Twitter via xurl CLI: post, search, DM, media, v2 API.

**Use when:** Automating Twitter posts, managing accounts, or building Twitter-based workflows.

**Key features:**
- Post tweets and threads
- Search tweets and users
- Direct messaging
- Media uploads (images, videos)
- Twitter v2 API integration
- CLI-based automation

**Use cases:**
- Automated posting schedules
- Social media monitoring
- Engagement automation
- Content distribution
- Bot accounts management

---

## Quick Start

```bash
# Post tweet
/xurl "Post: Just deployed v2.0! 🚀"

# Search
/xurl "Search for tweets about AI agents"

# DM
/xurl "Send DM to @user: Thanks for the feedback!"
```
