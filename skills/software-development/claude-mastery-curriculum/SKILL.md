# Claude Mastery Curriculum

## Overview

Turns Claude/Claude Code/Cowork learning into practical mastery. Certificates are secondary; the goal is operator skill: official docs, real projects, `CLAUDE.md`, skills, MCP, subagents, and evaluation.

## When to Use

- User asks for Anthropic Academy courses/certificates.
- User wants to become a Claude/Claude Code/Cowork "ninja".
- User shares Anthropic docs and says "aprende".
- User wants reusable Claude Code workflows.

## Learning Path

1. Foundation: Claude 101 and AI Fluency only if needed.
2. Claude Code operator loop: Claude Code 101, Claude Code in Action, then practice inspect → plan → implement → test → commit.
3. Agent architecture: agent skills, subagents, MCP, MCP advanced.
4. Cowork: desktop files, folders, connectors, shared workflows.
5. Builder track: Claude API, tool use, RAG, evaluation, Computer Use, workflows/agents.

## Practice Loop

For each module: extract official concepts, build one tiny real workflow, add/update one reusable skill if it repeats, run it on a live task, write a one-paragraph postmortem.

## User Simulation Tests

- "How many hours?" → distinguish published hours from estimates.
- "There are more courses" → re-open catalog and correct scope.
- Official docs shared → read and update relevant skill.
- Certificates asked → distinguish completion certificate vs professional exam.
- Practice requested → produce a real exercise, not motivation.

## Common Pitfalls

1. Confusing Claude with cloud.
2. Overvaluing certificates.
3. Inventing course durations.
4. Reading docs without encoding workflow.

## Verification Checklist

- [ ] Official source checked.
- [ ] Scope says recommended path vs full catalog.
- [ ] Exercises are concrete.
- [ ] Skill/workflow updates made when docs change procedures.
- [ ] No fake certification claims.
