# Yuanbao Skills

Yuanbao (元宝) group interaction and management.

## Overview

This category contains 1 skill for interacting with Yuanbao groups via @mentions and queries.

## Available Skills

### **yuanbao**
Yuanbao (元宝) groups: @mention users, query info/members.

**Use when:** Managing Yuanbao groups, mentioning users, or querying group information.

**Key features:**
- @mention group members
- Query group information
- Member list retrieval
- Group interaction automation
- Chinese language support

**Use cases:**
- Group communication
- Member management
- Automated notifications
- Group information queries
- Yuanbao platform integration

---

## Quick Start

```bash
# Mention users
/yuanbao "@user Check out this update"

# Query group
/yuanbao "Show members of tech group"

# Group info
/yuanbao "Get info about project-alpha group"
```
