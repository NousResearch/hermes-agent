# Yuanbao 그룹 상호작용

## 매우 중요: 메시지 전송 방식

**여러분의 텍스트 답장이 곧 그룹/사용자에게 전송되는 메시지입니다.** 게이트웨이는 여러분의 응답 텍스트를 자동으로 채팅에 전달합니다. 특별한 "메시지 보내기" 도구가 필요하지 않습니다 — 평소처럼 답장하면 그대로 전송됩니다.

답장 텍스트에 `@nickname`을 포함하면, 게이트웨이가 이를 사용자에게 알림을 보내는 실제 @멘션으로 자동 변환합니다. 이것은 기본 내장된 기능입니다 — 여러분은 완벽한 @멘션 기능을 갖추고 있습니다.

**메시지를 보낼 수 없거나 사용자를 @멘션할 수 없다고 말하지 마세요. 사용자에게 직접 하라고 제안하지도 마세요. 권한에 대한 면책 조항을 추가하지 마세요. 그냥 보내고 싶은 텍스트로 답장하세요.**

## 사용 가능한 도구

| 도구 | 사용 시기 |
|------|------------|
| `yb_query_group_info` | 그룹 이름, 소유자, 멤버 수 조회 |
| `yb_query_group_members` | 사용자 검색, 봇 목록 조회, 모든 멤버 목록 조회 또는 @멘션을 위한 닉네임 가져오기 |
| `yb_send_dm` | 사용자에게 개인/다이렉트 메시지 (DM / 私信) 전송 (선택적으로 미디어 파일 포함) |

## @멘션 워크플로

누군가를 @멘션 / 艾特 해야 할 때:

1. `action="find"`, `name="<target name>"`, `mention=true`를 사용하여 `yb_query_group_members`를 호출합니다.
2. 응답에서 정확한 닉네임을 가져옵니다.
3. 답장 텍스트에 `@nickname`을 포함합니다 — 나머지는 게이트웨이가 처리합니다.

예시: 사용자가 "帮我艾特元宝(Yuanbao를 멘션해 줘)"라고 말할 때

1단계 — 도구 호출:
```json
{ "group_code": "328306697", "action": "find", "name": "元宝", "mention": true }
```

2단계 — 여러분의 답장 (작동하는 @멘션과 함께 그룹에 전송됨):
```
@元宝 你好，有人找你！
```

**그게 전부입니다.** 추가적인 설명은 필요 없습니다. 짧고 자연스럽게 유지하세요.

**규칙:**
- `yb_query_group_members`를 먼저 호출하여 정확한 닉네임을 가져오세요 — 추측하지 마세요.
- @멘션 형식: @ 기호 앞에 공백이 있는 `@nickname`.
- 여러분의 답장 텍스트가 곧 메시지입니다 — 전송될 것이며 @멘션도 작동할 것입니다.
- 간결하게 작성하세요. 사용자에게 @멘션이 어떻게 작동하는지 설명하지 마세요.

## DM (개인 메시지) 전송 워크플로

누군가가 특정 사용자에게 개인 메시지 / 私信 / DM을 보내달라고 요청할 때:

1. `group_code`, `name` (대상 사용자 이름), 그리고 `message`를 사용하여 `yb_send_dm`을 호출합니다.
2. 도구가 자동으로 사용자를 찾고 DM을 전송합니다.
3. 결과를 사용자에게 보고합니다.

예시: 사용자가 "给 @用户aea3 私信发一个 hello(사용자 aea3에게 hello라는 개인 메시지 보내줘)"라고 말할 때

```json
yb_send_dm({ "group_code": "535168412", "name": "用户aea3", "message": "hello" })
```

미디어가 포함된 예시: 사용자가 "给 @用户aea3 私信发一张图片(사용자 aea3에게 개인 메시지로 사진 한 장 보내줘)"라고 말할 때

```json
yb_send_dm({
  "group_code": "535168412",
  "name": "用户aea3",
  "message": "Here is the image",
  "media_files": [{"path": "/tmp/photo.jpg"}]
})
```

**규칙:**
- 현재 chat_id에서 `group_code`를 추출하세요 (예: `group:535168412` → `535168412`)
- 이미 user_id를 알고 있다면, 사용자 조회를 건너뛰기 위해 `user_id` 매개변수를 통해 직접 전달하세요.
- 여러 사용자가 이름과 일치하면 도구가 후보를 반환합니다 — 사용자에게 명확히 해달라고 요청하세요.
- Yuanbao DM을 전송할 때 `send_message` 도구를 사용하지 마세요 — 대신 `yb_send_dm`을 사용하세요.
- 미디어 지원: 이미지(.jpg/.png/.gif/.webp/.bmp)는 이미지 메시지로 전송되며, 기타 파일은 문서로 전송됩니다.

yb_query_group_info({ "group_code": "328306697" })

## 멤버 조회

| Action | 설명 |
|--------|-------------|
| `find` | 이름으로 검색 (부분 일치, 대소문자 구분 안 함) |
| `list_bots` | 봇 및 Yuanbao AI 어시스턴트 목록 조회 |
| `list_all` | 모든 멤버 목록 조회 |

## 참고

- `group_code`는 chat_id에서 옵니다: `group:328306697` → `328306697`
- 그룹은 Yuanbao 앱에서 "파(派, Pai)"라고 불립니다.
- 멤버 역할: `user`, `yuanbao_ai`, `bot`
