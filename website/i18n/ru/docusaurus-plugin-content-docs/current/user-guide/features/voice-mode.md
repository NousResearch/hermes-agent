# Режим голоса

Voice mode позволяет говорить с Hermes голосом и получать голосовые ответы в CLI, Telegram и Discord. Для локального STT/TTS путь остаётся максимально простым: Hermes старается работать без лишних обязательных ключей и при этом поддерживает более качественные облачные провайдеры, если они нужны.

## Что нужно заранее

- рабочая установка Hermes Agent
- корректно выбранный провайдер модели
- микрофон и устройство вывода звука
- при желании: ключи для облачного TTS или STT

## CLI voice mode

### Быстрый старт

```bash
hermes voice
```

### Как это работает

- Hermes слушает микрофон
- speech-to-text превращает речь в текст
- текст отправляется в обычный агентный цикл
- ответ озвучивается через TTS

### Практические настройки

- для локального STT можно использовать `faster-whisper`
- для TTS подойдут локальные или облачные варианты
- если микрофон или аудиоустройство не видны, сначала проверьте системные разрешения

## Голосовые ответы в gateway

Hermes может отвечать голосом и в мессенджерах, если соответствующая платформа это поддерживает. На практике чаще всего это Telegram и Discord.

### Telegram

В Telegram голосовой режим удобен для коротких заметок, быстрых вопросов и hands-free сценариев.

### Discord

В Discord голосовой режим особенно полезен в серверных каналах и voice channels, где Hermes может не только писать текст, но и отвечать голосом.

## Что важно для качества

- STT должен быть достаточно точным
- TTS должен звучать естественно и не “рубить” интонацию
- в каналах с шумом полезна более строгая настройка фильтрации тишины

## Конфигурация

Основные настройки обычно лежат в `config.yaml` и `.env`.

Типовые параметры:
- выбор STT-провайдера
- выбор TTS-провайдера
- чувствительность детектора тишины
- параметры голосового канала для Discord

## Устранение неполадок

- если звук не захватывается, проверьте разрешения ОС
- если TTS молчит, проверьте выбранный провайдер и аудиовывод
- если распознавание даёт мусор, попробуйте другой STT-провайдер или более чистый микрофонный вход

## Когда voice mode особенно полезен

- при работе без клавиатуры
- в мессенджерах
- для быстрых проверок и диктовки
- в ситуациях, где удобнее говорить, чем печатать

## Дальше

После базовой проверки voice mode можно подключать к gateway-платформам, настраивать качество TTS и переходить к полноценным голосовым workflow.
