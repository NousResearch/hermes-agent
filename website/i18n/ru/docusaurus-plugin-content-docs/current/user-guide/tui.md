# TUI

TUI — это современный терминальный интерфейс Hermes Agent: модальные overlay, удобный ввод, подсветка и более живой ритм работы по сравнению с классическим CLI.

## Запуск

```bash
hermes --tui
```

Полезные варианты:

```bash
hermes --tui
hermes --continue
hermes --continue --tui
hermes --tui --run
```

## Зачем он нужен

TUI полезен, когда хочется:
- быстрее работать с несколькими сообщениями подряд
- видеть состояние сеанса без лишнего шума
- пользоваться мышью и визуальными подсказками
- не терять привычный CLI-рабочий процесс

## Что нужно для работы

TUI использует тот же Hermes runtime, что и классический CLI. Отдельной “магии” настраивать не нужно:
- выберите модель через `hermes model`
- убедитесь, что сессии и конфиг работают
- при желании подстройте `hermes config`

## Ключевые сочетания клавиш

Поддерживаются базовые действия:
- `Enter` отправляет сообщение
- `Alt+Enter` или `Ctrl+J` вставляет новую строку
- `Ctrl+C` прерывает текущую задачу
- `Esc` закрывает overlay или возвращает фокус в обычный режим

## Slash-команды

Slash-команды работают так же, как в CLI:

| Команда | Назначение |
|---|---|
| `/help` | Показать доступные команды |
| `/model` | Переключить модель |
| `/tools` | Настроить инструменты |
| `/personality` | Выбрать персону |
| `/save` | Сохранить разговор |

## Встроенный рендеринг

TUI поддерживает аккуратный вывод математических формул, code blocks и длинных ответов. Это особенно полезно, если вы часто работаете с кодом, промптами и техническими заметками.

## Busy indicator

Если Hermes занят, TUI показывает статус выполнения и помогает понять, когда можно отправлять следующий запрос.

## Сессии

TUI хранит ту же модель сессий, что и CLI. Можно:
- продолжить последний диалог
- открыть конкретную сессию
- вернуться к классическому CLI в любой момент

## Когда всё же выбрать CLI

Если нужен максимально простой текстовый поток без интерфейсных слоёв, классический CLI остаётся отличным выбором. TUI — это надстройка, а не замена.
