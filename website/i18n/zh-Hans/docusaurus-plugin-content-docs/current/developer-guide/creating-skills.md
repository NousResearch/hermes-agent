# 创建技能

Skill 是可分发的 Markdown 工作流包。通常包含 `SKILL.md`、前置条件、步骤和示例。

## 要点

- 明确目标和适用场景
- 尽量小而专注
- 提供清晰的安装与使用说明

## 配置设置（config.yaml） {#config-settings-configyaml}

在技能前置说明中，建议明确需要的 `config.yaml` 配置项、默认值和最小可运行配置，方便用户快速落地。
