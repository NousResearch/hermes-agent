# 使用 Cron 自动化

Hermes Cron 可以按固定时间、每隔一段时间或一次性计划运行任务。它既支持普通 LLM 任务，也支持仅脚本任务。

## 你可以做什么

- 定时告警
- 每日简报
- 周报 / 月报
- 外部 API 轮询
- 仅脚本 watchdog
