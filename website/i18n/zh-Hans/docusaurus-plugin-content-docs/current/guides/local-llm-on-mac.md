# 在 Mac 上运行本地 LLM

本页介绍如何在 macOS 上为 Hermes Agent 准备本地大模型环境。

通常需要确认以下几点：

- 你的模型服务已经启动，并提供可访问的 API 端点。
- Hermes 的 `base_url` 和模型名称已正确配置。
- 本机算力、内存和上下文长度满足你的使用场景。

如果你准备在本机长期使用本地模型，下一步通常是把模型服务固定为后台进程，写好 Hermes 的 `base_url` 与模型名，然后按实际负载调高超时、上下文和并发限制。

## 超时与稳定性 {#timeouts}

本地模型首 token 延迟通常更高。若你频繁遇到超时，请先提高请求超时，再观察模型加载与推理峰值期间的稳定性。
