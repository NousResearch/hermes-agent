# Teams Meetings

这套功能会把 Teams 会议事件转成 Hermes 任务，再把摘要或动作项投递到指定目的地。

## 相关操作

- 验证 webhook 与订阅
- 续期 Graph subscription
- 检查 token 健康
- 运行会议流水线运维脚本
