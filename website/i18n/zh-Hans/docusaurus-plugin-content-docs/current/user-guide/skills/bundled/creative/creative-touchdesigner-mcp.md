# TouchDesigner MCP 集成

此页面概述如何使用 Hermes 的 MCP（多客户端进程）能力来与 TouchDesigner 交互：

- 运行 TouchDesigner 时通过 MCP 将渲染参数和事件发送给 Hermes。
- 使用 MCP 插件在渲染管线中触发脚本、生成场景描述并回传资产。
- 示例：通过 `/skill touchdesigner render <preset>` 在远程渲染机上触发渲染并把结果回传到会话。

后续将补充示例命令、配置项和最佳实践。
