# G0DM0D3 — 安全研究技能

本页只保留高层次说明，用于介绍该技能在模型安全评估中的用途。

## 用途

- 评估模型在提示注入、越狱与拒答边界上的表现。
- 帮助安全研究人员验证防护、过滤与响应策略。
- 对比不同模型在相同测试集下的稳定性。
